# JobSheets

A clean, minimal job application tracker built with vanilla HTML, CSS, and JavaScript. Track applications, statuses, and dates — all saved locally in your browser.

**Live demo:** https://jobsheets.vercel.app

---

## About

JobSheets is a lightweight job application tracker built with vanilla JavaScript. Add applications with a company name, role, and status, and track your progress across Applied, Interview, Offer, and Rejected stages. All data persists in localStorage so nothing gets lost on refresh. Built as a real tool to solve a real problem — tracking co-op and internship applications during recruiting season.

---

## Features

- Add and delete job applications
- Color coded status badges — Applied, Interview, Offer, Rejected
- Summary counts showing totals at a glance
- Auto date stamping on every entry
- localStorage persistence across sessions
- Clean dark UI with a purple color palette

## Getting Started

To run locally:

1. Clone the repo
```bash
   git clone https://github.com/roleyrogan/JobSheets.git
```
2. Open `index.html` in your browser

No dependencies or installation required.

## Roadmap

JobSheets is actively being improved. Planned features for upcoming versions:

- Filter applications by status
- Edit status without deleting and re-adding
- React rebuild for a more dynamic UI
- Backend with Node and Express for server-side data storage
- PostgreSQL database to replace localStorage
- User authentication so your applications sync across devices
